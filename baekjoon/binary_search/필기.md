# 문제에서 이분탐색 힌트는 보통 이거다.
- “정적 배열에서 존재 여부를 여러 번 물어보는데, 크기가 커서 선형 탐색이 안 된다.”


# 1. 일반 이분탐색

- 정렬된 배열에서 특정 값이 있는지 찾기


# 2 . 파라메트릭 서치

- 정답이 될 수 있는 값들 중에서 조건을 만족하는 최댓값/최솟값 찾기

예:

랜선 자르기: 만들 수 있는 최대 길이
나무 자르기: 가져갈 수 있는 최대 절단 높이
입국심사: 모두 심사받는 최소 시간


# 판단 기준
일반 이분탐색 냄새
- 정렬된 배열
- 값 존재 여부
- 위치 찾기
- 개수 찾기

예:

수 찾기
숫자 카드 2


파라메트릭 서치 냄새
- 최대값/최솟값 구하라
- 가능한가? 불가능한가? 판별 가능
- 조건 만족 여부가 단조적

예:

랜선 자르기
나무 자르기
입국심사
공유기 설치


#### 일반 이분탐색: target을 찾는다
#### 파라메트릭 서치: mid가 답이 될 수 있는지 검사한다
